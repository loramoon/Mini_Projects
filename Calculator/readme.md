### Tkinter Calculator with the following functions:
- summation
- subtraction
- multiplication
- division
- integer division
- percentage calculation
- chain calculation
- clearing the result
- limiting the displayed result to 10 decimal places (if the result is float with more than 10 digits after the decimal point
- error handling
